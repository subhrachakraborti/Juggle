# **App Name**: Juggle

## Core Features:

- Secure Data Integration: Connect bank accounts via Plaid API and integrate Google Calendar for a unified view of finances and commitments.
- AI-Powered Financial Forecasting: Employ Gemini, Claude, or another LLM API to analyze financial data and calendar events for cash flow projection, identifying potential shortfalls using LangChain.
- Personalized Financial Advice: Provide specific, actionable recommendations based on the user's financial habits and upcoming commitments using AI tool, such as suggesting micro-savings plans.
- Real-time spending categorization: Categorize expenses into meaningful categories, so they are easier to track
- Dashboard: Display key metrics, upcoming bills, and a calendar view for financial events and recommendations.
- Gig worker focus: Provide insights that will better serve a volatile income with no safety net, and no employer benefits.
- User Authentication: Secure user authentication using Firebase Auth.

## Style Guidelines:

- Primary color: Teal (#008080) to evoke trust and financial security.
- Background color: Light teal (#E0F8F8), a desaturated shade of the primary color to provide a calm backdrop.
- Accent color: Lime green (#BFFF00), an analogous color, to draw attention to key actionable items and recommendations.
- Headline font: 'Space Grotesk', sans-serif, for a techy, scientific feel, since gig workers can appreciate bleeding-edge technology. Body font: 'Inter', sans-serif, since longer text may appear on some dashboard screens
- Use clear, minimalist icons to represent transaction categories and financial actions.
- Design a clean, intuitive dashboard layout that provides an overview of financial status and upcoming events.
- Incorporate subtle animations to indicate data loading, successful transactions, or new recommendations.
